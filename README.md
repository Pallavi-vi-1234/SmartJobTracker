# Smart Job Tracker 🚀

A MERN stack project to track job applications.

## Features
- Add Jobs
- Save jobs in MongoDB
- Display saved jobs
- Frontend + Backend connection

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas

## Run Project

Frontend:
```bash
npm run dev
```

Backend:
```bash
node server.js
```
